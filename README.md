**SayaLulus** adalah layanan informasi kelulusan peserta didik. Sebelumnya dikenal dengan [Graduation4](https://github.com/HilmiZul/graduation2021) yang dibuat pada tahun 2021 untuk lingkungan SMKN 4 Tasikmalaya dan terus mengalami perubahan.

### Fitur
- Countdown pengumumnan
- Info kelulusan
- Pengaturan Umum
- Pengaturan Kelulusan:
  - Umum
  - Kelulusan: No.SK, tanggal putusan dan waktu pengumumnan
  - Teks pernyataan peserta
- Upload/replace daftar peserta
- Kelola peserta

Lihat [Changelog](https://github.com/HilmiZul/sayalulus/blob/main/CHANGELOG.md) untuk daftar perubahan.


# Setup

```bash
# npm
npm install

# bun
bun install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# deno
deno task dev

# bun
bun run dev
```

### Production

```bash
# npm
npm run build

# bun
bun run build
```

Locally preview production build:

```bash
# npm
npm run preview

# bun
bun run preview
```
